## ğŸ”§ Troubleshooting for Beginners

### "Python is not recognized"
- Install Python from https://python.org
- **Must check** "Add Python to PATH" box
- Restart computer after installation

### "Permission denied" or "Access denied"
- Right-click `QUICK_START.bat`
- Choose "Run as administrator"

### App doesn't open in browser
- Look for a web address in the black window (like http://localhost:8501)
- Copy and paste it in your browser

### "Models not found" error
- Run `QUICK_START.bat` again
- Wait for training to complete
